# 🍅 Dancing Tomato Club 

“가볍지만 하루를 돌아보게 만드는 공간."

하루에 한 번 나를 칭찬하는 커뮤니티입니다. 주제에 따라 서로 칭찬·공감하며 소소한 소통을 주고받을 수 있습니다.

## 👩🏻‍💻 팀원 
BE/ 권소영 (1인)

## 🌱 주요 기능
### 📝 셀프 칭찬 글쓰기
- 자유롭게 하루를 돌아보고 칭찬 글을 작성할 수 있는 기능
- 해시태그로 글의 성격을 분류
- 이미지 업로드 기능

### 📚 피드 / 무한 스크롤
- 카드 레이아웃으로 게시글을 한눈에 확인
- IntersectionObserver로 자연스러운 무한 스크롤 구현
- 좋아요·댓글 상태 반영
- 로그인 사용자의 좋아요 여부에 따른 UI 동기화

### 🧑‍🤝‍🧑 프로필 / 회원 기능
- 회원가입 및 세션 기반 로그인
- 프로필 이미지·닉네임 수정
- 내가 쓴 글 / 좋아요한 글 모아보기(개발 예정)

### 🛠 관리자 페이지
- 전체 회원 목록 조회
- 회원 삭제 기능 제공
- 전체 게시글 모니터링 가능

## 🧩 테크 스택 
- HTML / CSS 
- JavaScript
